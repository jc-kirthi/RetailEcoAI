# 🌿 RetailEcoAI

An AI-powered customer segmentation system that helps sustainable retail businesses identify customer groups and recommend personalized marketing strategies using Machine Learning.

## 📌 Project Overview

RetailEcoAI is a Machine Learning application designed to classify customers into different sustainability-focused segments based on their purchasing behavior.

The project combines:

- K-Means Clustering
- Decision Tree Classification
- Interactive Gradio Dashboard

to provide real-time customer segmentation and marketing recommendations.

---

## 🎯 Problem Statement

Retail businesses often struggle to identify customers interested in sustainable products.

RetailEcoAI analyzes customer purchasing patterns and predicts the customer's sustainability segment, enabling businesses to deliver targeted marketing campaigns and improve customer engagement.

---

## ✨ Features

- AI-powered customer segmentation
- Real-time prediction using Decision Tree

- Marketing strategy recommendation
